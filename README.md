## Installation
```txt
npm i discord.js-selfbot-alex
```

## Delcaration
```js
const Alex = require('discord.js-selfbot-alex')
```

## Login (Token)
```js
Alex.login(token)
```

## Code Demo
```js
const Alex = require('discord.js-selfbot-alex');
Alex.setStatus("idle");
Alex.joinVoice("")
Alex.login(token)
```
