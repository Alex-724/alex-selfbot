## Installation
```txt
npm i djs-selfbot-alex
```

## Delcaration
```js
const Alex = require('djs-selfbot-alex')
```

## Initialising (Mandatory)
```js
Alex.login(token)
```

## Code Demo
```js
const Alex = require('djs-selfbot-alex');
Alex.setStatus("idle");
Alex.joinVoice("")
Alex.login(token)
```
